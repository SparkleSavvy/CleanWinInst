Ultimate Windows Unattended Installation & Customization Script

Deploy a clean, fresh, and highly customized Windows installation — completely unattended. Forget about tedious post-installation tweaking.

Key Features

Bloatware Removal

  • Prevents installation of unwanted built-in apps
  • Removes optional Windows Features
  • Removes system Capabilities

Privacy & Security

  • Disables Windows Defender during installation
  • Blocks Bing integration and telemetry
  • Disables SmartScreen and Fast Startup

Visual & UI Customization

  • Sets Dark Theme for System and Apps
  • Configures Taskbar and Start Menu
  • Uses Classic Context Menu and shows File Extensions

System Optimization

  • Enables Long File Paths support
  • Allows PowerShell Scripts to run
  • Bypasses TPM and Secure Boot checks

User & Account Setup

  • Creates local administrator account
  • Enables Auto-Logon for first boot
  • Applies customizations to Default User profile

Regional & Language

  • Primary UI Language: English (US)
  • Additional Keyboard Layout: Russian
  • Time Zone: Russian Standard Time

How to Use

  1. 1

    Download the `unattend.xml` file.

  2. 2

    Place it in the root directory of your Windows installation media (USB drive or extracted ISO folder), alongside the `sources`, `boot`, etc., folders.

  3. 3

    Boot from the media. The Windows Setup will automatically detect and apply all configurations.

  4. 4

    Sit back, relax, and watch your perfectly customized Windows install itself!

Important: The script includes a command to download and execute a script from `https://get.activated.win` during the first user logon. This is typically used for activation. Use this only if you trust the source. You can easily remove or modify this command in the XML if needed.
